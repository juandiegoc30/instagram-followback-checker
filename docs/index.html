FAQ

Common questions

Does IFC ask for my Instagram password?

No. It uses the active Instagram session already available in your browser.

Why can Instagram rate-limit actions?

Instagram may temporarily restrict repeated automated actions. IFC includes paced execution modes, but excessive use can still be limited by Instagram.

Can I export the results?

Yes. You can copy the list or export it as CSV or JSON.

Why is IFC not on the Chrome Web Store?

This first release is distributed as an unpacked extension while the project is being validated and improved.

Does IFC send my data to a server?

No. The analysis runs in your browser. Exports are generated locally only when you request them.

Is IFC affiliated with Instagram?

No. IFC is an independent open-source project and is not affiliated with Instagram, Meta, or their products.

Is it safe to unfollow many accounts at once?

Bulk actions can trigger temporary Instagram limits. IFC includes slower execution modes, but you should use the feature carefully.
